# Getting Started

### Node Setup

This app was built with node `v20`. If you are using [nvm](https://github.com/nvm-sh/nvm) you can switch the correct node version:

```
$ nvm install && nvm use
```

Install node dependencies:

```
$ npm i
```

Start the dev server:

```
$ npm run dev
```
